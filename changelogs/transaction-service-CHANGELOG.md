# Changelog - Transaction Service

All notable changes to the Transaction Service will be documented in this file.

## [0.0.1] - 2025-11-08
### Added
- Initial transaction service with wallet management
- Database support with SQLX
- RESTful API for wallet CRUD operations
- Wallet model with support for multiple currencies
- Integration with auth service via gRPC for JWT validation
- Database migrations for wallets table
- Comprehensive logging throughout the service
